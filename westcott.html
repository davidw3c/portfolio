Westcott House (2020)

Working closely with our wonderful client Shelby, I built the entirety of the front end of the Westcott House website to be both stylish and user-friendly. We worked through page-by-page to ensure the clients requirements were met.

The website was built using SCSS with flexbox for layout, vanilla Javascript and PHP for Wordpress.

A number of animations feature on the website that I coded using Javascript. For example, the homepage slideshow features a ‘slide timer’ which I hand-coded, and many pages contain quotes which fade in, which uses some Javascript to detect when the quote element is visible on page scroll.

With as much attention to detail as possible in the front end using SCSS, I also fully integrated the website into Wordpress with a custom-built ‘page builder’ that utilises Advanced Custom Fields. This includes relationship pickers for related pages & footer quotes, sidebar featured images & quotes, repeater fields for download data tables. There are forms for example for the newsletter signup, as well as many different custom post groups.
