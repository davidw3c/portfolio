Case Studies: UK Music

UK Music (2019)

The UK Music website was built to be as accessible as possible. The raw HTML view includes a ‘skip to content’ link which is visually hidden instead of using ‘display:none’, and HTML sections with specific ID’s and headings. Body text contrast ratio achieves a AAA rating.

Page landmarks have the correct ARIA roles assigned to them.

You can use the keyboards tab key to browse through the website. The browser default ‘outline’ has not been hidden using CSS. There is a Javascript powered accessible navigation whereby dropdown menus can be accessed via the tab key.

Articles include the ability to upload audio transcripts of the article using a dedicated online text to voice MP3 service.

Footer links give users the choice of whether to click on the mailto email link or whether to highlight the text without clicking a link.
