Terminal Commands Cheat Sheet

Please note this article is under construction. For Linux / Mac OS Command Line - Become a Terminal Nerd!

Command Name Stands for What it does Example
cd Change Directory Changes from one directory to another. To go 'up one level' you can type: cd .. cd "path/to/directory"
ls List Directory View the contents inside the current directory in an inline format ls
ls -la List Directory View the contents inside the current directory in a readable list format ls -la
chmod Change Permissions change the access permissions of file system objects ls -la
open Open file Type this command, followed by a space and the file name (including file extension) you wish to open and the file shall open. open filename.txt
exit Exit Shell End whatever you are currently doing. exit
