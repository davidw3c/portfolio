Okra AI

In this case study I will demonstrate how I coded an interactive 'Brain' element using vanilla Javascript which allows users to hover over the regions of an SVG generated brain image to view popup info windows as well as alter the background of the 'hero' element according to what colour zone is interacted with.

Includes a video case study as well as images and code snippets.

December 2022

The Ogden Trust

A central theme of the new Ogden Trust website has been to achieve good Accessibility. To meet this aim, The Ogden Trust hired the RNIB to produce reports throughout the build. showing how we could create the most accessible website possible.

These reports highlighted a number of improvements I made, including responsive & clearly labelled data tables, use of accessibility friendly labels on forms and buttons, correct alt tag descriptions on images, adding ARIA states and landmarks to the HTML, good colour contrast and many more.

Mid 2022

Westcott House

Working closely with our wonderful client Shelby, I built the entirety of the front end of the Westcott House website to be both stylish and user-friendly. We worked through page-by-page to ensure the clients requirements were met.

The website was built using SCSS with flexbox for layout, vanilla Javascript and PHP for Wordpress.

Mid-Late 2020

UK Music

The UK Music website was built to be as accessible as possible. The raw HTML view includes a ‘skip to content’ link which is visually hidden instead of using ‘display:none’, and HTML sections with specific ID’s and headings. Body text contrast ratio achieves a AAA rating.

The website was built using SCSS with flexbox for layout, vanilla Javascript and PHP for Wordpress.

Mid-Late 2019

Domino Printing (Email Campaign)

From late 2018 to present (January 2019), I was tasked with hand coding a number of email templates for Domino printing. These email templates ranged from standard marketing emails, to internal newsletters.

Domino used an online email marketing automation tool called Salesforce Pardot.

This tool allows you to take your HTML email templates, and set up editable 'regions' so that when Domino staff members want to create a new email campaign, they can use your HTML template and simply edit the content, without any coding knowledge required themselves.

Nov 2018 - Jan 2019

The Woolf Institute

The Woolf Institute wanted a brand new website to coincide with their move to an impressive new premises on Madingley Road in Cambridge.

I integrated the website into Craft CMS, and used Javascript and SASS to create a number of unique touches and add interactivity to the website. For example, the book logo pages light up when hovering over the logo in the header, and also reveal when a user scrolls to the footer (using Javascript for viewport detection).

Lucy Cavendish College

I was tasked with building the Front End of the Lucy Cavendish website, and also with integrating the website into a Content Management System. In this case, we used Expression Engine.
